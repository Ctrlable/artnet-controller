# Ctrlable Art-Net (DMX) for Home Assistant

Home Assistant integration for **Art-Net / sACN / KiNET** DMX lighting, set up
entirely in the UI (no YAML) with a visual **Art-Net Patch** panel.

This is the **distribution** repository consumed by HACS. Sources are maintained
privately in `Ctrlable/ha-artnet-led-src`.

## Install (HACS)

1. **HACS → ⋮ → Custom repositories** → add `Ctrlable/ha-artnet-led`,
   category **Integration**.
2. Install **Ctrlable Art-Net (DMX)** → **restart Home Assistant**.
3. **Settings → Devices & Services → Add Integration → "Ctrlable Art-Net"**
   (Scan or Manual), then open the **Art-Net Patch** sidebar panel.
